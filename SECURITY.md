# Security policy

Do not open a public issue containing a vulnerability, credential, token, authorization code, cookie,
callback URL with query parameters, complete claim payload, or personal data.

Report security concerns privately through the security reporting channel published by the
`mydatum-ai` GitHub organization. Include only the affected starter and version, safe reproduction
steps, expected behavior, and redacted error class. MyDatum will acknowledge the report, assess
supported versions, and publish remediation or withdrawal guidance when appropriate.

The current starter release and the immediately preceding minor release receive compatibility fixes.
Critical security fixes may withdraw an older release without a transition period.
